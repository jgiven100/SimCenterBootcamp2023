# Day 3 Assignments

## Example 1

### Compile
```
g++ file3.cpp
```

### Run
```
./a.out name.txt name.dat
```
where `name` is either `small` or `big`. The output binary file may be checked with
```
$ wc small.dat
    1   2 160 small.dat
$ wc big.dat
    32   657 32000 big.dat
```
where `160` and `32000` are expected.
