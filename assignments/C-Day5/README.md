# Day 5 Assignments

## Example 1: Vector Norm

### Compile
```
mkdir build
cd build/
cmake ..
make
```

### Update Threads on TACC
Change the number of thread on Frontera using
```
export NUM_OMP_THREADS=N
```
where `N` is the number of threads. 

### Run
```
./hello
```

```
./normVector N
```
where `N` is the size of the vector.

```
./normVector2 N
```
where `N` is the size of the vector.
